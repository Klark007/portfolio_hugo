These are some small recordings of games I made before and during the first year of University using Godot and is mostly here for nostalgia. Two of the entries were game jam games while the other was a game I worked on for a few months during highschool. This first game was made all the way back in Godot 2 using GDScript (python like syntax) and was a platformer with multiple boss fights. This one read levels from image files as the Tile set editing in Godot 2.x wasn't as well implemented as it is nowadays.
